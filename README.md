# Site Web E‑Commerce

## Description
Ce projet est un site web de e‑commerce développé en Terminale, utilisant les langages HTML, CSS et JavaScript.  
L’objectif était de concevoir une plateforme simple et fonctionnelle permettant la présentation de produits, la gestion d’un panier et une expérience utilisateur fluide.

## Contexte académique
Ce projet a été réalisé en Terminale dans le cadre d’un exercice de développement web,  
avec pour objectif de mettre en pratique les langages HTML, CSS et JavaScript.


## Fonctionnalités
- Page d’accueil avec présentation des produits
- Catalogue interactif avec fiches détaillées
- Interface responsive adaptée aux ordinateurs et mobiles
- Design ergonomique avec mise en page en CSS

## Technologies utilisées
- HTML5 : structure des pages
- CSS3 : mise en forme et design
- JavaScript : gestion des interactions et du panier
